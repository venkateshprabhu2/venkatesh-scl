## Installation
```bash
    git clone <url>
    git branch -b <new_branch_name> #to add a branch
    git branch --list #to view all the branch
    git branch #to view current branch
    git status #to check status
    git checkout <branch_name> #to change the branch
    git add <filename> #to add a file to git
    git commit -m "message" #to commit
    git push origin <branch_name>


```
